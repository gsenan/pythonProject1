 # AGE CHECKER:
 A simple python script that checks if a given age is that of an adult or a minor.

 # USAGE:
 1. Set the 'age' variable to the desired age.
 2. Run the script.

# EXAMPLE:
```age = 25

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
```

# Output:
```You are an adult.```
